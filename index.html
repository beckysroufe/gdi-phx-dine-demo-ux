What is UX?

What is UX?

UX == User Experience

  • We're all about the users.
  • Practice of using methods and techniques to research what users need to design products and services through them
  • Used to describe the effect created by interactions and perceptions that someone has when using a product or service

Confused about UX?

  • UX: User Experience
    • Encompasses any and all interactions between a user and a product, service, etc.
    • Example: Design a house by examining who lives in it, how they'll use it, and create foundation and structure
  • UI: User Interface
    • Focuses on taking UX and translating it into the look and feel
    • Example: Design how the house looks, gives it an overall feel and creates interaction details

Clearing up more UX confusion

  • Lots of roles + terms: user experience (UX) designer, user interface designer, UX architect, web designer, user researcher, interaction designer, information architect, UX architect, graphic designer, visual designer, content strategist ... and so on

Main UX Focus

On a day-to-day basis, user experience may (and should) touch all realms but is primarily under user research and user experience design.

Brief History of UX

Early 1900's

Frederick Winslow Taylor
Mechanical Engineer

  • Authors “The Principles of Scientific Management”
  • Research into efficiency of interactions between workers and tools

Henry Ford
Founder of Ford Company

  • Pioneers mass-production techniques
  • Shape what interactions between laborers and tools should be

1948: Toyota

  • Similar to Ford in valuing efficiency in engineering and production
  • Sought employees' input - valued “respect for people” philosophy
  • Workers involved in troubleshooting processes they were a part of via Andon Cord

1955: Henry Dreyfuss

Henry Dreyfuss
Industrial designer

  • Wrote "Designing For People"
  • Described many of the methods that UX designers employ today to understand and design for user needs
“When the point of contact between the product and the people becomes a point of friction, then the industrial designer has failed. On the other hand, if people are made safer, more comfortable, more eager to purchase, more efficient - or just plain happier - by contact with the product, then the designer has succeeded.”

1970: Xerox PARC

  • PARC (Palo Alto Research Center)
  • Yields graphical user interface (GUI), mouse, and computer-generated bitmap graphics
  • Influenced the first commercial GUI in Apple’s Macintosh

1990s - Today

  • 1990's: Apple brings design and UX into prominence
  • 1995: Don Norman coins term user experience to describe his team's work at Apple
  • 2007: The iPhone puts user experience in limelight by delighting users

UX Basics

UX Basics

  • RESEARCH
  • DESIGN
  • IMPLEMENT

UX Basics: Research

Find out who your users are!

  • Guerrilla research (interview a small number of target users)
  • External studies and research data
  • Personas, User Stories, Journey Mapping

UX Basics: Design

Build a system to fit user needs

  • Information Architecture (IA)
  • Process diagrams & wireframes
  • Design comps
  • Interaction specs
  • Style + pattern guides
  • Prototypes

UX Basics: Implement

  • Testing, testing, testing!
  • Collaboration
  • Analytics

UX Rules To Live By

  • Keep user informed
  • Speak the users' language
  • User control and freedom
  • Consistency and standards
  • Error prevention
  • Help users recognize, diagnose, and recover from errors
  • Recognition rather than recall
  • Flexibility and efficiency of use
  • Aesthetic and minimalist design
  • Help and documentation

UX in Today's Tech Teams

UX in the industry

  • Careers: UX architect, UX designer, UI designer, visual designer, UX researcher, UI developer
  • Still fairly new for most teams to adopt and have dedicated resource(s)
  • Often teams want a "unicorn" - someone to do it all!
  • Pay the UX debt upfront - incorporate UX at the beginning + throughout projects
